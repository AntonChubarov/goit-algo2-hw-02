# goit-algo2-hw-02

## GOIT Design and Analysis of Algorithms Homework 2

### Requirements

- Python 3.10+
- Git

### Usage

Clone this repository:
```shell
git clone https://github.com/AntonChubarov/goit-algo2-hw-02.git
```

Navigate to project directory:
```shell
cd goit-algo2-hw-02
```

Run unit tests:
```shell
python3 -m unittest
```
